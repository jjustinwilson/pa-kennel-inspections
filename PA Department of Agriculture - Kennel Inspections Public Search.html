Kennel Inspection Search Instructions

The Pennsylvania Kennels licensed with the Dog Law Enforcement Office can be searched from this web page.

The following information contains the details of the searching functionality available:
  • County: Search based on the kennel county.
  • Kennel Type: Search based on the kennel type.
  • License Number: Search based on kennel license number.
  • Kennel (Business) Name: Search based on kennel business name.
  • City: Search based on the kennel city.
  • Zip code: Search based on the kennel zip code.
Searches are not case sensitive.

NOTE: Kennel inspections are available in Adobe's PDF format for viewing and printing.
